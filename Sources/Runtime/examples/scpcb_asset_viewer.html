About

SCPCB Asset Viewer is a web-based demo that showcases the Blitz3D-WASM runtime for loading and rendering SCP: Containment Breach assets in the browser.

Built with Three.js and WebAssembly, this viewer can load SCPCB's proprietary RMesh room format along with textures and models.

Controls:
• Left-click + drag: Rotate
• Right-click + drag: Pan
• Scroll: Zoom
• WASD / Arrow Keys: Move
