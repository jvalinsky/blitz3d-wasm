# `deploy.sh` archived (2026-02-02)

`deploy.sh` has been archived to `docs/archive/dated/2026-02-02_deploy.sh`.

Reason: it relied on `sudo` + mutating `/var/www/html` and ad-hoc `npx` behavior.

Preferred flow now:

- Build: `deno task web:build` (writes repo-root `dist/`)
- Serve: `deno run -A web/serve_dist.ts` (or any static file server pointed at `dist/`)
