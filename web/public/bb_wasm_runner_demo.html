BB Runner Demo (WASM-only, worker + watchdog)

This is the “safe runner” half of a web interpreter: it runs a compiled BB→WASM module in a Web Worker, stubs missing imports, captures output, and can terminate on infinite loops (timeout).

Inputs

Run instantiates in a dedicated worker and calls the first available entrypoint: main, _start, Main, Main_, __main.
To compile quickly from the repo, use:
deno run -A Tools/bb_deno_compile_and_run.ts path/to/program.bb
Then upload the produced .wasm.

Output

idle
